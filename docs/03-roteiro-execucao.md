## Plano B / Contingências

| Problema potencial | Solução alternativa |
| ------------------- | ------------------- |
| Internet indisponível | Demonstrar TalkBack offline (funciona sem internet); usar vídeo salvo localmente para demonstrar legendas |
| Projetor não funciona | Demonstração individual ou em pequenos grupos diretamente pelo celular |
| Menos participantes que o esperado | Adaptar para atendimento mais individualizado, aprofundando cada ferramenta |
| Mais participantes que o esperado | Dividir em dois grupos: visual e auditivo, cada um com 3 integrantes do grupo de apoio |
| Participante sem smartphone | Usar um dos celulares do grupo como emprestado para a prática |
| Dificuldade de comunicação com deficiente auditivo | Utilizar comunicação escrita (papel/celular) e material impresso com instruções |
